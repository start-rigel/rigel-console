Rigel Console

AI Hardware
Briefing

我们把用户需求收成结构化 `user_request`，再由 build-engine 结合当前型号级价格清单输出一份可读的推荐结果。当前这版 UI 对齐的是文档里已经定下来的字段，不额外扩范围。

Request user_request 预算、用途、品牌偏好、特殊要求、补充说明，和文档字段完全一致。
Route POST /catalog/recommend Console 保持薄壳，推荐生成仍然交给 build-engine。
Output summary / parts / total 结果区直接对应当前 API 契约里的结构化推荐输出。
推荐结果会显示在这里。
